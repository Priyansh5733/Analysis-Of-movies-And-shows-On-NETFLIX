Netflix Movies & TV Shows Analysis
Project Overview
This project performs an **Exploratory Data Analysis (EDA)** on Netflixâ€™s catalog of movies and TV shows to identify trends related to content type, release year, age certification, genres, and production countries.

The analysis is conducted using **Python**, with the help of **pandas** for data manipulation and **matplotlib** for visualization.

 Objectives
- Compare the number of **Movies vs TV Shows**
- Analyze **age certification distribution**
- Study **content release trends over the years**
- Identify **top content-producing countries**
- Compare yearly trends of **movies and TV shows**
- Analyze **genre distribution** after cleaning multi-genre data
